What the Dashboard Shows (The Cool Stuff)
The Big Numbers: It instantly shows total money made, the average amount customers spend per order, and the total number of pizzas sold.

The Rush Hours: It maps out which days of the week and what times of the day are the busiest so the restaurant knows exactly when they need extra staff.

The Favorites: It breaks down which pizzas sell the most (like Veggie vs. Pepperoni) and which sizes (Small, Medium, Large) are the most popular.

How I Built It
Cleaned the Data: I opened the CSV file in Tableau and fixed any messy data—like making sure the computer understood what was a date, a time, or a price.

Created the Visuals: I built clean charts and graphs instead of boring tables.

Made it Interactive: I set it up so you can click on things. For example, if you click on "Friday," the whole dashboard changes to show you only what people buy on Fridays.
